# Trading App

This is a FastAPI-based Trading Application.

## Features
- User authentication with JWT
- Redis caching
- API routes for operations and tasks
- Role-based access control

## Requirements
- Python 3.8+
- FastAPI
- Redis
- PostgreSQL (or any other database)
